All icons have a 32x32px bound as default format. They can be resized in our 16px baseline as required.

To use an icon, import it (by one of the names below) from `@magnetis/astro-galaxy-icons` and use it as a React component.

You can modify an icon's size by using the prop `size` with one of the following values: `24`, `32` or `40`. You can also modify the color with the prop `color`. See the props table at the bottom of this page for details.
