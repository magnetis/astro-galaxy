## radio buttons

Radio buttons appear when the user must select only one option from more than two.

You must create a unique `id` for each radio button shown in the same page.

Mouseover to view `:hover` state.

To create the disabled state, add the disabled attribute on the `Radio` component.

<Playground
  scope={{ RadioWrapper, Radio, RadioShape, Label }}
  code={`
    <RadioWrapper>
      <Radio id="radio1" name="radiogroup1" />
      <RadioShape />
      <Label htmlFor="radio1">Radio</Label>
    </RadioWrapper>
    <RadioWrapper>
      <Radio id="radio2" name="radiogroup1" defaultChecked />
      <RadioShape />
      <Label htmlFor="radio2">Selected</Label>
    </RadioWrapper>
    <RadioWrapper>
      <Radio disabled id="radio3" name="radiogroup1" />
      <RadioShape />
      <Label htmlFor="radio3">Disabled</Label>
    </RadioWrapper>
  `}
/>

Default label behavior is top alignment. You can pass the `center` prop to the `RadioWrapper` component to switch to center alignment.

<Playground

## toggles

A simple control that is used to quickly switch between two possible states.

Mouseover to view `:hover` state.

To create the disabled state, add the `disabled` attribute to the `Toggle` component.

<Playground
  isPlaygroundWithFragment={false}
  scope={{ Toggle, useState }}
  code={`
    () => {
      const [isToggled, setToggle] = useState(false);
      return (
        <>
          <Toggle label="toggle me" isToggled={isToggled} onClick={() => setToggle(!isToggled)} />
          <Toggle disabled label="disabled" />
        </>
      )
    }
  `}
/>

## sliders

Sliders give users a visual indication of adjustable content, mostly numerical (for time, money, etc).
They can be used with or without text labels.

Adjust `min` and `max` attributes to set the value range.
If you need to manipulate the slider handle position, use the `value` attribute.
You must also set an unique `id` for each slider in your page.

Sliders contains built-in label and counter label components.
To define the label use the `label` attribute and for counter label, `counterLabel` attribute.
If you don't want to display the Slider labels, just ignore the two attributes.

To display the gradient full fill, set `fullFill` property to `true`.

To set the disabled state, add the `disabled` attribute.

<Playground
  isPlaygroundWithFragment={false}
  scope={{ Slider, useState }}
  code={`
    () => {
      const [value, setValue] = useState(2500);
      const counterLabel = value + " Coins";
      return (
        <>
          <Slider label="Wallet" counterLabel={counterLabel} value={value} max={5000} onChange={(e) => setValue(e.target.value)}/>
          <Slider id="full-slider" label="Full Fill" fullFill={true} />
          <Slider id="disabled-slider" label="Disabled" disabled />
        </>
      )
    }
  `}
/>
