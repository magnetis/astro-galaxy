<Text font="secondary" fontSize="large" color="moon400" marginBottom={5}>
  Buttons define what action will occur when the user interacts with features and components of the
  platform. Buttons are used to initialize and dictate the actions on the interface, either in the
  background or foreground of an experience.
</Text>

<Text as="h2" fontSize="large" color="moon500">
  primary buttons
</Text>

Use these for the main action in a section or screen. You should display only one primary button at a time.

To create a primary button, import Button component and use `variant` prop with `primary.uranus` value. You can replace `uranus` with other set of colors. Check the table below.

<Playground
  scope={{ Button }}
  code={`
    <Button variant="primary.uranus">
      uranus
    </Button>
    <Button variant="primary.earth">
      earth
    </Button>
    <Button variant="primary.venus">
      venus
    </Button>
    <Button variant="primary.mars">
      mars
    </Button>
    <Button disabled>
      disabled
    </Button>
  `}
/>

</Text>
