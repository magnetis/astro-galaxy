## text area inputs

Text areas are used in contexts that demand long and descriptive texts.

Focus the input to view `:focus` state.

To create the disabled state, add the disabled attribute on the `TextArea` component.

<Playground
  scope={{ TextAreaWrapper, TextAreaLabel, TextArea }}
  code={`

  `}
